# oh-my-claude

Claude Code 설정과 레퍼런스 자료를 모아둔 저장소입니다.

## 구성

### 1. [oh-my-claudecode](./oh-my-claudecode/)

[oh-my-claudecode(OMC)](https://github.com/Yeachan-Heo/oh-my-claudecode) 플러그인 소스 스냅샷입니다.
에이전트, 커맨드, 스킬, 문서 등 OMC의 전체 구성을 포함합니다.

### 2. [ECC (everything-claude-code)](./ECC/)

[everything-claude-code(ECC)](https://github.com/affaan-m/ECC) 소스 스냅샷입니다.
언어별 코딩 룰, 에이전트, 스킬, 훅, 워크플로우 등 Claude Code용 종합 설정 모음입니다.

### 3. [superpowers](./superpowers/)

[superpowers](https://github.com/obra/superpowers) 소스 스냅샷입니다.
브레인스토밍, TDD, 체계적 디버깅 등 프로세스 중심 스킬 모음 플러그인입니다.

### 기타

- [result/](./result/) — 실제 `~/.claude`에 적용한 설정 백업 (agents, rules, skills, CLAUDE.md)
  - [result/every-thing/](./result/every-thing/) — ECC의 언어/프레임워크별 룰 전체 복사본 (typescript, python, java, react 등 22종)
